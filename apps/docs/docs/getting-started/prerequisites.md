# Prerequisites

Before you start, make sure you have:

- `Node.js >= 20`
- `npm >= 10`
- A running PostgreSQL database
- Git (recommended)

Optional (for production/distributed setups):

- Redis (if you use `@meridianjs/event-bus-redis` or `@meridianjs/job-queue-redis`)
