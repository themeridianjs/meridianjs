REST API for MeridianJS — an open-source modular project management framework.

## Authentication
All `/admin/*` endpoints require a valid JWT passed as a Bearer token:
```
Authorization: Bearer <token>
```
Obtain a token via `POST /auth/login`. The `/auth/*` and `/public/*` endpoints are unauthenticated.

## Roles
The JWT payload carries a `roles` array. Built-in roles:
- `super-admin` — full access to all workspaces and data
- `admin` — full access within their workspace
- `member` — filtered by workspace and project membership

Custom fine-grained permissions are managed via the AppRole module.
